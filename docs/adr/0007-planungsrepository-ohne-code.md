# ADR-0007: Planungsrepository ohne Code

- Status: ersetzt durch ADR-0009
- Datum: 2026-07-13

## Kontext

Vor der Implementierung sollen Produktumfang, Datenmodell und Architektur stabilisiert werden.

## Entscheidung

Dieses private Repository enthält in der aktuellen Phase ausschließlich Planung, Architektur, Recherche und Notizen. Anwendungscode wird hier zunächst nicht angelegt.

## Folgen

- klare Trennung von Denken und Implementieren
- Entscheidungen bleiben nachvollziehbar
- kein voreiliger Technologieaufbau
- später muss entschieden werden, ob dieses Repository weiterverwendet oder ein finales Produktrepository erstellt wird
