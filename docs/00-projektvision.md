# Projektvision

## Problem

Tropico-6-Spieler finden Produktionsverhältnisse häufig in einzelnen Guides, Tabellen oder Diskussionen. Diese Informationen beantworten jedoch meist nur theoretische Fragen. Sie bilden selten den konkreten Inselstand, räumliche Gruppen, Arbeiterbedarf, Lagerprobleme, Prognosen oder Diagnosefälle gemeinsam ab.

## Vision

Eine leicht bedienbare, lokal speichernde Webanwendung soll Spielern ermöglichen, ihre Produktionswirtschaft der Kolonialzeit realitätsnah zu erfassen und Ausbauentscheidungen gefahrlos zu testen.

## Produktversprechen

Der Nutzer kann:

- seinen aktuellen Inselstand erfassen
- Gebäude frei zu Clustern gruppieren
- Produktionsketten automatisch auswerten
- mehrere Ausbauprognosen erzeugen
- Ist-Stand und Prognosen vergleichen
- Rohstoff-, Arbeiter- und Transportengpässe erkennen
- Beobachtungswerte aus dem Spiel zur Diagnose verwenden
- seine Projekte lokal speichern, exportieren und wieder importieren

## Erfolgsmerkmale

Das Produkt ist erfolgreich, wenn ein Spieler innerhalb weniger Minuten:

- eine Insel anlegen kann
- ein Cluster mit Gebäuden erfassen kann
- eine Produktionskette auswerten kann
- eine Prognose erstellen kann
- eine verständliche Engpassaussage erhält

## Zielgruppe

- neue Tropico-6-Spieler
- Spieler, die Produktionsketten optimieren möchten
- Spieler mit größeren oder räumlich getrennten Produktionsgebieten
- Spieler, die mehrere Ausbauvarianten vergleichen möchten
- Community-Mitglieder, die Daten prüfen und ergänzen möchten
