# Entscheidungsprotokoll

## 2026-07-13

- Erste Version fokussiert ausschließlich die Kolonialzeit.
- DLC-Inhalte werden grundsätzlich berücksichtigt.
- Direkte Exportgüter werden zusätzlich zu echten Verarbeitungsketten aufgenommen.
- Cluster sind frei zusammenstellbare Gebäudegruppen.
- Gebäude können gruppiert oder einzeln erfasst werden.
- Entfernung ist optional und wird bei leerem Wert ignoriert.
- Warenentfernung und Arbeiterentfernung werden fachlich getrennt.
- Ist-Stand bleibt erhalten; Prognosen verändern ihn nicht.
- Mehrere Prognosen und Vergleiche gehören zum Kernumfang.
- Local-First-Webanwendung wird gegenüber Excel bevorzugt.
- Speicherung erfolgt zunächst lokal im Browser.
- Export und Import sind verpflichtend.
- GitHub Pages ist als spätere statische Bereitstellung vorgesehen.
- Backend, Benutzerkonten und Cloud-Synchronisierung werden für Version 1 ausgeschlossen.
- Dieses Repository bleibt zunächst privat und enthält keinen Anwendungscode.
